# Karpathy-GPT

In this project, I'll be following along with the LLM Legend himself, Andrej Karpathy.  He has a YouTube video ["Let's build GPT: from scratch, in code, spelled out."](https://www.youtube.com/watch?v=kCc8FmEb1nY_) where I get to follow along and build an LLM from start to finish using the Shakespear dataset.

# What is this for?
I believe that people who understand how things work are much better off at using whatever that 'thing' is. Cars, games, home appliances, websites; we benefit to learn how things work.  

With the insane growth in AI, I not only don't want to be caught lagging behind. I want to be the person who can explain how it works in detail. This will make me better at prompting, finding use cases, and other ways of interacting with LLMs today. 

It's going to be exciting.
